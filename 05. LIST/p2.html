Order list or Number list types

1) Numberic Number (1, 2, 3, 4..n) and this type id default type for order list

2) Capital Roman number (I, II, III)

3) Small roman number (i, ii, iii)

4) Capital Alphabet(A, B, C, D)

5) Small Alphabet (a, b, c, d)

we can change the type of order list or number list using type attribute
