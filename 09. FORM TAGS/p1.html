HTML FORM TAG

the form tag is used to collect diffrent user data

Form with attributes

Action attribute
server path/server file path/name/url address
method
  1. get => data will be visible (getting data from the server)
  2. post => data will be hidden (sending form data to server)
noValidate
if we dont want to validate the form taht time we can use novalidate attribute
enctype
  1. urlencoded is the default formate
  2. multipart/form-data => if the form includes input type file
autocomplete
on (autofill will be on) and off => autofill will be off => it is used to provide the autocompletion for form fields
form tags
  1. fieldset => this tag is used to add a border to form tags
  2. legend => this tag is used to give caption or title for fieldset tag
  3. input tag => types=> text, email, number, password, radio, checkboxes, date, datetime, datetime-local, time, month, week, file, search, color, tel(deprecated), range, submit, reset, hidden, image, url
  4. label => this tag is used to provide the label for inputs
  5. select => this tag is used to create a dropdowns in website to select single or multiple options
  6. textarea => this tag is used to enetr the multiple lines of text like address information
  7. button => submit, reset, button > is used to submit or reset the form data
